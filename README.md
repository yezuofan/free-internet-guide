## ⚠️ 重要提醒

1. **不要一次性充值太多**：机场有跑路风险，最多充3个月用量
2. **不要只看价格**：中转线路晚高峰降速70%，便宜不等于好用
3. **至少备2个机场**：一个主力 + 1个备用，永远有 Plan B
4. **优先选 IEPL/IPLC 专线**：不过境、不抢带宽，晚高峰依然坚挺
